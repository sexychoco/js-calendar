## js-calendar
Build a calendar with HTML, CSS, javascript

### Description
HTML,CSS,Javascript 를 활용하여 캘린더를 만들어보았습니다.
new Date()를 적극적으로 사용하였으며 for문을 통한 자동 생성 기능을 연습하였습니다.
